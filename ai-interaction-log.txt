             ═══════                                          ══════

             1. User visits /contact
                ──── GET /contact ─────────────────────►  pageRouter: res.sendFile('contact.html')

                → loads /css/styles.css (static)

             4. contact.js intercepts submit
                → e.preventDefault()

                ──── POST /contact ────────────────────►  pageRouter.post('/contact')
                     Content-Type: application/json             │
                     { name, email, message }                   ├── validates fields
                                                                ├── logs to console
                ◄──── JSON response ──────────────────── ├── sends { success, message }

             5. contact.js reads response
                → injects message into #form_response
                → resets form if success
